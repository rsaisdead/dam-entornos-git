# Resumen de Programación

En este módulo hemos aprendido los conceptos **básicos y avanzados** del desarrollo de software. 

Ha sido un proceso *muy interesante* donde hemos pasado de la teoría algorítmica a la práctica escribiendo nuestro propio código.

> "El buen código es su propia mejor documentación."

## Herramientas Utilizadas

* **Git**: Sistema de control de versiones.
  * Manejo de ramas (*branches*) como `master` y `nueva`.
  * Guardado del historial mediante *commits*.
* **Java**: Lenguaje de programación.
  * Uso de variables y bucles.
  * Manipulación de Arrays (ordenar, buscar, filtrar).

### Pasos para nuestro flujo de trabajo

1. Entender el problema planteado.
2. Escribir y probar el código en Java.
3. Guardar la versión usando Git.
